# Vogue-Vault---Shopping-Application
Vogue Vault is an e-commerce web application designed to offer a smooth and user-friendly online shopping experience. Customers can explore products, add items to their cart, register and log in securely, and place orders efficiently.

The application uses PHP for backend development, MySQL for database management, and HTML, CSS, and JavaScript to create a responsive and interactive user interface.

✨ Features
User Registration and Login
Secure User Authentication
Product Catalog
Product Categories
Product Search
Shopping Cart
Add to Cart / Remove from Cart
Order Placement
Order Summary
Responsive Design
Database Integration with MySQL
